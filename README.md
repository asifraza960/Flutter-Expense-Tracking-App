# 💰 Expense Tracker App (Flutter)

A modern Expense Tracking Mobile Application built with Flutter that helps users manage their daily expenses efficiently. The app allows users to record,
categorize, and monitor their spending using a clean and user-friendly interface.
# ✨ Features

➕ Add new expenses easily

📊 Track daily spending

🗂 Categorize expenses

💾 Local data storage using SharedPreferences

⚡ Fast and responsive UI

🧠 State management with BLoC

🏗 Built using Clean Architecture
# 🛠 Tech Stack

Flutter

Dart

BLoC (Business Logic Component)

SharedPreferences (Local Storage)

Clean Architecture

Material UI
